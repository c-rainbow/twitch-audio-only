크롬에서 트위치 방송을 라디오 모드로 들을 수 있게 해주는 크롬 확장 프로그램입니다. 

설치 후 스트리머의 채널에 가시면 방송화면 왼쪽 아래에 라디오 모양의 아이콘을 확인하실 수 있습니다. 클릭하시면 라디오 모드가 실행되고, 한번 더 클릭하시면 라디오 모드가 종료됩니다.

문제 해결: 라디오 모드를 사용 중 문제가 생겼을 경우 페이지를 새로고침 후 다시 시도해 주세요.

깃헙 주소: https://github.com/c-rainbow/twitch-audio-web


Release notes
v0.5.1 (2022-07-12)
- 광고가 재생되는 동안 라디오 모드가 15초정도 무음이었던 버그를 수정하였습니다.
- 일부 사용자에게서 라디오모드가 실행되지 않던 버그를 수정하였습니다.

v0.5.0 (2022-07-06)
- 스트리머의 채널 페이지를 연 지 20분이 지나면 라디오모드를 실행할 수 없던 버그를 수정하였습니다.
- 코드 리팩터를 하였습니다.
- manifest v3으로 업데이트하였습니다.

v0.4.0 (2022-05-07)
- 트위치의 DOM 변경으로 인해 라디오모드 버튼이 보이지 않는 버그를 수정하였습니다.
- 툴팁 메세지가 복구되었습니다.
- 문제가 있을 시 페이지를 새로고침하라는 안내를 툴팁에 넣었습니다.
- 모든 의존성 패키지를 업데이트하였습니다.

v0.3.1 (2021-05-17)
- Hotfix: 최근 트위치의 CSS 변경으로 인하여 라디오 모드 버튼의 툴팁 메세지가 항상 보이게 되는 버그가 생겼습니다. 일단은 모든 툴팁 메세지를 삭제하였습니다

v0.3.0 (2021년 4월 4일)
- 확장 프로그램이 사용하는 라이브러리를 모두 최신 버전으로 업데이트
- 문제가 생겼을 시 페이지를 새로고침 후 다시 시도해달라는 메세지 추가

v0.2.0 (2020년 12월 18일)
